Introduction

UniAspect is:

- An Aspect-Oriented Programming framework for multiple languages.

- Implemented based on UNICOEN, and provides the unified description of an aspect for multiple languages.

- Able to deal with concerns cross-cutting over multiple languages as a single aspect.

Usage

Execution Environment

UniAspect and UNICOEN are developed with C#. Therefore, you need to prepare .NET framework to execute UniAspect. Visual Studio 2010 is also required when you want to build UniAspect from source code. Please install these tools if needed.

All dlls is required when you invoke UniAspect. Please confirm /bin directory when the system does not work well.
